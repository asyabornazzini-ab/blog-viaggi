# Il blog di viaggi di Asya

Questo è il "deposito" dei file del mio blog di viaggi.
Il sito è costruito con [Astro](https://astro.build/) e pubblicato gratuitamente su Cloudflare Pages.

## Struttura del progetto

- `src/pages/` — le pagine del sito (per ora solo la home)
- `src/layouts/` — il "vestito" comune a tutte le pagine
- `src/styles/` — i colori, i font e l'aspetto generale del sito
- `public/` — il favicon (l'icona piccola nella scheda del browser) e, in futuro, le immagini

## Come funziona

Ogni volta che modifico un file qui dentro, Cloudflare ricostruisce il sito da capo e mette online la nuova versione, in modo automatico.
